# dotfiles

Automating the personal environment

* bash
* fish
* git
* gpg2
* Homebrew
* Hyper
* iTerm2
* Ruby
* ssh
* Vim

## Installation

  1. Clone the repo

      ```sh
      $ git clone --recursive https://github.com/arbourd/dotfiles
      ```

  1. Run `dot.sh`

      ```sh
      $ ./dot.sh
      ```

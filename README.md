# dotfiles

A collection / backup of my dotfiles

* Atom
* fish-shell
* git
* gpg
* iTerm2
* Vim

## Installation

  1. Clone the repo

    ```bash
    $ git clone --recursive git://github.com/arbourd/dotfiles
    ```

  1. Run `bootstrap.sh`

    ```bash
    $ ./bootstrap.sh
    ```

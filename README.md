# dotfiles

A collection / backup of my dotfiles

* fish-shell
* git
* gpg2
* Homebrew
* Hyper
* iTerm2
* Ruby
* Vim

## Installation

  1. Clone the repo

      ```sh
      $ git clone --recursive git://github.com/arbourd/dotfiles
      ```

  1. Run `bootstrap.sh`

      ```sh
      $ ./bootstrap.sh
      ```
